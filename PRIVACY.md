# Privacy Policy - Typing & Time Counter

**Last updated:** July 17, 2026

## Data Collection
This plugin collects and stores the following data **locally on your machine only**:
- Keystroke count (total characters typed)
- Active coding time duration
- Break reminder settings and preferences
- User-initiated break check-ins ("Done resting")

## Data Storage
- All data is persisted in your IDE's local configuration directory (`~/.config/JetBrains/*/options/`)
- **No data is transmitted to any external server or third-party service.**
- No analytics, tracking, or telemetry of any kind.

## Third-Party Services
None. This plugin does not connect to any network services.
